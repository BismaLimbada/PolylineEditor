PolyLine Editor: Phase 1 - Requirements

User Scenarios (What is Wanted)

1. The Modification Scenario: A user draws a shape but needs to adjust one corner to align with a grid. The 'Move' requirement allows this without redrawing.

2. The Correction Scenario: A user accidentally adds an extra vertex. The 'Delete' requirement allows for targeted removal to maintain drawing integrity.

Functional Requirements

  • Begin ('b'): Creates a new polyline entry in the data array.
  • Delete ('d'): Removes the vertex closest to the mouse pointer.
  • Move ('m'): Repositions the nearest vertex to the current mouse coordinates.
  • Refresh ('r'): Clears the canvas and redrafts all stored polylines (Max: 100).

Controls: [b] Begin | [m] Move | [d] Delete | [r] Refresh | [q] Quit
